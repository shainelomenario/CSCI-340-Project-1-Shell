# CSCI-340-Project-1-Shell

First project that's a Linux Shell implemented in C. 

## Features
- Basic Command Execution
- Internal Commands
- I/O Redirection
- Unidirectional Piping

## Instructions
Compile the code
``` gcc CSCI340Shell.c -o shell```
Run 
```./shell```
Enter any commands, few examples are from test.txt
